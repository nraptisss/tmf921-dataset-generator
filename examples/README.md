# TMF921 Intent Examples

This directory contains reference TMF921 Intent examples used during the development and validation of the dataset generator.

## Files

- **IB1_Construction.json** - Basic construction intent example
- **IR1_1_Construction.json** - Resource layer PropertyExpectation example with latency/throughput
- **IS1_Construction.json** - Service layer DeliveryExpectation example

These examples are from the TMF921 Intent Management specification and serve as reference implementations for the Turtle RDF structure and TMF ontology usage.

## Source

Examples are based on the TMF921 Intent Management API v5.0.0 specification.
